1. Запустить hyper-v, зайти на сервер через mobaxsterm
2. ip br -a, для того чтобы узнать Ip
3. Перейти по ip:8080, на этом порту крутиться jenkins
4. Запустить job, freestyle - интерфейсный метод, но запуск контейнера через командную строку docker run -p 81:80 svaxel/docjen
5. Запустить job, pipeline - декларативный подход, сам запускать на 81 порту контейнер
